# День 2. Основы Vue и реактивности

Темы:
- создание Vue-приложения
- реактивные переменные
- computed-свойства
- интерполяция текста в шаблоне.

## Практическое задание

В этом уроке ты создашь свой рейтинг любимых книг.

Цель задания:
- научиться выводить списки в шаблоне vue
- освоить работу с реактивными переменными

## Порядок выполнения

1. Сделай форк учебного репозитория в github https://github.com/rsukhar/startup-hacker-course#  
2. склонируй проект на локальную машину и перейди в папку day_2  
2. создай в day_2 vue-проект, следуя инструкциям в официальной документации  
3. ! каждое следующее задание нужно выполнять в папке day_N, где N-номер дня стажировки. Т.к. задания построены вокруг одного проекта, код, написанный в прошлый день нужно будет переносить в новую папку.  

4. создай компонент главной страницы приложения  
5. объяви реактивную переменную со списком книг (массив с 3-5 объктами)  
6. объекты должны хранить следующую информацию:  
 - название  
 - описание  
 - жанр  
 - обложка книги  
 - рейтинг  
7. выведи данные в template с помощью v-for (не забудь добавить атрибут key)  
8. выведи рейтинг в виде звездочек (лучше использовать svg-иконки).
