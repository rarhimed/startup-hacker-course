# День 6. Экосистема и стандарты разработки

Темы:  
- Vue Router  
- библиотека startup-ui  

## Практическое задание
В этом уроке мы добавми еще одну страницу в приложение, настроим роутинг и подключим UI-библиотеку.  

Цель задания:
- научиться пользоваться Vue Router  
- познакомиться с библиотекой startup-ui  

## Порядок выполнения

1. Создай компонент страницы «О нас» (можно заполнить любым текстом-рыбой)  
2. Добавь возможность переключаться с главной страницы на «О нас» с помощью Vue Router  
3. UI наших проектов построен на библиотеке startup-ui, поэтому важно уметь с ней работать и быстро находить нужные компоненты. В качестве знакомства с библиотекой замени все html-элементы и собственные vue-компоненты на компоненты startup-ui.
