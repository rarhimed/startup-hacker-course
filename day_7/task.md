## Задания

### Таймер для стрима. 

Напиши приложение «Таймер», которое показывает время, оставшееся до начала стрима.

Требования: 
 - дисплей показывакт время в формате «ДД ЧЧ : мм : сс»  (дни, часы, минуты, секунды)
 - Под кол-вом дней, часов, меясяцев и секунд расположена подпись в правильном склонении. Например «2 дня 5 часов...», «1 день 3 часа...» и т.д.  
 - дисплей обновляется каждую секунду
 - цифры обновляются плавно, с применением анимации  
 - если время стрима наступило, на дисплее выводится надпись «В эфире».

При необходимости можно пользоваться сторонними библиотеками.

### Мемори

Напиши игру «Мемори». 

На экран выводятся карточки с числами, расположенные шапкой вверх. Пользователь нажимает на любую карточку, она открывается, затем на вторую. Если числа на карточках совпали, карточки удаляются. Если не совпали — обе карточки переворачиваются шапкой вверх. 
 
Игра завершается, когда на экране не останется ни одной карточки.

Требования: 
- после удаления пары карточек, расположение других карточек не должно меняться
- сверху выводится блок со статистикой:
  - совершено ходов
  - осталось карточек
  - времени прошло.
 
Дополнительно: 
2.1 добавь возможность выбирать уровень сложности (например, 12, 24, 36 карточек). Информацию об уровне сложности выводи в блоке со статистикой.  
2.2 после окончания игры выводи оценку результата в зависимости от кол-ва совершенных ходов.  


### Rocket science

В этом задании мы будем запускать космические корабли! 

Что нужно сделать:  
3.1 На экране выводим фон с космосом (ты найдешь его, как и ассеты космических кораблей, в папке public)  
3.2 Слева — меню для выбора типа корабля  
3.3 У каждого типа корабля есть две постоянные характеристики (установи их на своё усмотрение):  
    - объем топлива  
    - скорость перемещения (чем быстрее движется корабль, тем быстрее расходуется топливо).  

3.4 При выборе корабля должно открываться модальное окно с полем имени корабля и кнопкой «Запустить».Запускать корбали без имени нельзя.  
3.5 Добавим интерактивности:  
- когда уровень топлива достигает 10% от объема бака, корабль начинает снижаться. Чтобы корабль продолжил летать, его нужно заправить — для этого пользователь кликает по нему мышкой.  
- когда уровень топлива достигает 0, корабль падает и разбивается. Для этого состояния есть соответствующие ассеты в space/public.  

3.6 Выводи статистику:
- кол-во активных кораблей  
- кол-во разбитых кораблей   
- общее кол-во запущенных кораблей.
