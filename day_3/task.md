# День 3. Знакомство с Vue3 (события, связывания)

Темы:  
- обработка событий  
- привязка классов и стилей  
- работа с данными формы  
- v-model  

## Практическое задание
В этом уроке мы сделаем рейтинг книг интерактивным.  

Цель задания:  
- научиться выводить списки в шаблоне vue  
- освоить работу с реактивными переменными  
- научиться привязывать классы и стили  
- освоить работу с формами  

## Порядок выполнения
1. Сделай звездочки интерактивными: звезда, на которую кликнул пользователь и все звезды до нее, загораются желтым цветом. Остальные — становятся серыми.  
2. Добавь в верхнем углу обложки большую звезду со значением рейтинга. Если рейтинга нет — серую звезду с прочерком  
3. Добавь кнопку «Добавить», по нажатию на которую открывается форма создания новой книги  
4. Форма должна содержать следующие поля:  
 - название (text)  
 - описание (textarea)  
 - обложка (text)  
 - жанр (выпадающий multiple-список)  
 - 18+ (чек-бокс)  
5. После добавления книги, форма должна быть очищена  
6. Добавь возможность редактировать карточки с книгами. Важно, чтобы поля формы при открытии были заполнены данными карточки.  
7. Добавь кнопку «Отменить» которая очищает и закрывает форму.  
8. Добавь возможность удалять книги  

## Дополнительное задание

1. Добавь кнопку, которая сбрасывает рейтинг всех книг  
2. Добавь блок со статистикой, который содержит:  
 - суммарное кол-во книг  
 - средний рейтинг (число с плавающий точкой, две цифры после запятой).
